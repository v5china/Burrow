# Screenshots

Add clean captures here, then reference them from the top-level `README.md`.

Suggested set: `status.png` · `analyze.png` · `software.png` · `clean.png` ·
`optimize.png` · `hud.png`

Tip: capture each window over a plain wallpaper. Burrow's glass is
translucent, so whatever is behind the window shows through — a busy
background (other app windows) makes for messy screenshots.
